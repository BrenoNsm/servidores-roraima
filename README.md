# 📊 Pipeline de Dados - Servidores de Roraima

Sistema de coleta e análise de dados de servidores públicos do estado de Roraima.

##  Objetivo

Coletar, processar e analisar dados de servidores das esferas:
- ✅ Estadual (implementado)
- ⏳ Municipal (planejado) verificando a viabilidade!
- ⏳ Federal (planejado) verificando a viabilidade!

## 🚀 Instalação

```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/servidores-roraima.git
cd servidores-roraima

# Crie ambiente virtual
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# ou
.venv\Scripts\activate  # Windows

# Instale dependências
pip install -r requirements.txt
